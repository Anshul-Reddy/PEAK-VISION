# PEAK VISION

PEAK VISION is a study management platform built using Flask and MySQL.

## Features

* Notes Management
* CRUD Operations for Notes
* Question Bank
* Subject Filtering
* Progress Tracking
* Mark Questions as Completed

## Tech Stack

* Python
* Flask
* MySQL
* HTML
* CSS
* Jinja2

## Project Status

Version 1 (V1) Completed

## Future Enhancements (V2)

- AI-generated flashcards
- Weak subject prediction
- User authentication
- Dashboard analytics
- Personalized study recommendations
